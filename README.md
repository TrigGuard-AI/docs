# TrigGuard Documentation

Human-readable documentation explaining the TrigGuard protocol,
architecture, and ecosystem.

## Role in the Ecosystem

This repository contains explanatory material that helps developers
understand the protocol contract, architecture, and deployment models.

The canonical protocol definition lives in trigguard-protocol.

## Core terms

Use the same vocabulary as the protocol: **Execution Surface**, **Protocol Contract**, **Protocol Fingerprint**, **Audit Bundle**, **Verifier**, and **OER** (Operational Evidence Record).

## Authority Hierarchy

trigguard-protocol → canonical protocol specification  
trigguard-core-reference → reference implementation  
trigguard-js → SDK  
docs → explanatory documentation  
cloud → hosted runtime  
site → public website  
TrigGuard → demos and tooling
